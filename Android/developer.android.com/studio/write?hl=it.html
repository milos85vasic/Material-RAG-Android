Scrivi la tua app
Mantieni tutto organizzato con le raccolte Salva e classifica i contenuti in base alle tue preferenze.

Android Studio include strumenti per ogni fase dello sviluppo, ma la cosa più importante è scrivere l'app: scrivere il codice, creare layout, creare immagini ed essere produttivi lungo il percorso.

Questa sezione è incentrata su questi strumenti: gli strumenti che ti aiutano a scrivere la tua app e a scriverla rapidamente.

Produttività della programmazione

Le seguenti sono solo alcune delle funzionalità che ti aiuteranno a essere più produttivo durante la programmazione.

Completamento del codice

Il completamento del codice velocizza lo sviluppo delle app riducendo gli errori di digitazione e la necessità di cercare nomi di classi, metodi e variabili. L'editor di codice fornisce completamento di base, completamento intelligente e completamento delle istruzioni.

Scopri di più sul completamento del codice.

Creare modelli di completamento del codice personalizzati

I modelli pubblicati ti consentono di inserire snippet di codice per l'inserimento e il completamento rapidi di piccoli blocchi di codice. Per inserire un modello pubblicato, digita l'abbreviazione del modello e premi il tasto Tab. Android Studio inserisce lo snippet di codice associato al modello nel tuo codice.

Ad esempio, l'abbreviazione comp seguita da Tab inserisci il codice per una nuova funzione componibile. In alternativa, digita loge per trovare il metodo Log.e() e registra dal tuo codice.

Per visualizzare e personalizzare l'elenco dei modelli pubblicati supportati, fai clic su File > Impostazioni > Editor > Modelli live (Android Studio > Impostazioni > Editor > Modelli pubblicati su macOS).

Scopri di più sui modelli pubblicati.

Ricevi correzioni rapide da lint

Android Studio fornisce uno strumento di scansione del codice chiamato Lint che consente di identificare e correggere i problemi relativi alla qualità strutturale del codice, senza eseguire l'app o scrivere test.

Ogni volta che crei la tua app, Android Studio esegue Lint per controllare l'eventuale presenza di bug nei file di origine e cercare miglioramenti dell'ottimizzazione in termini di correttezza, sicurezza, prestazioni, usabilità, accessibilità e internazionalizzazione.

Scopri di più su Lint.

Visualizza la documentazione e i dettagli delle risorse

Puoi visualizzare la documentazione per un'API posizionando il cursore di testo sul nome del metodo/membro/della classe e premendo F1.

Le informazioni sono disponibili anche per altre risorse, come immagini e temi. Ad esempio, se posizioni il cursore sul nome del tema nel file manifest di Android e premi F1, potrai vedere la gerarchia di ereditarietà del tema e i colori o le immagini dei vari attributi.

Crea rapidamente nuovi file

Se vuoi creare un nuovo file, fai clic sulla directory che ti interessa nella finestra del progetto, poi premi Alt + Ins (Comando + N su Mac). Android Studio mostra una piccola finestra con un elenco di tipi di file suggeriti, in base alla directory selezionata.

Utilizzo delle risorse

Android Studio include le funzionalità e gli strumenti riportati di seguito per aiutarti a creare e gestire file di risorse.

Scopri di più sull'aggiunta di risorse.

Crea immagini per tutte le densità dello schermo

Android Studio include uno strumento chiamato Vector Asset Studio che ti aiuta a creare immagini che supportano ciascuna densità dello schermo. Puoi caricare il tuo file SVG per modificarlo o selezionarlo da una delle numerose icone di material design fornite da Google. Per iniziare, fai clic su File > Nuovo > Asset vettoriale.

Scopri di più su Vector Asset Studio.

Anteprima immagini e colori

Quando fai riferimento a immagini e icone nel codice, viene visualizzata un'anteprima dell'immagine nel margine sinistro per aiutarti a verificare il riferimento all'immagine o all'icona.

Per visualizzare l'immagine a grandezza originale, fai clic sulla miniatura nel margine sinistro. In alternativa, posiziona il cursore sul riferimento incorporato all'asset e premi F1 per visualizzare i dettagli dell'immagine, incluse tutte le dimensioni alternative.

Creare nuovi layout

Android Studio ti consente di visualizzare l'anteprima dei tuoi layout componibili quando utilizzi la funzione di anteprima componibile. Le anteprime delle componibili vengono visualizzate nella visualizzazione Design del file e vengono aggiornate in tempo reale man mano che le modifichi.

Se utilizzi layout XML, Android Studio ti offre l'editor di layout per visualizzare l'anteprima del layout durante la modifica del file XML.

Traduzione di stringhe UI

Lo strumento Editor delle traduzioni offre un'unica visualizzazione di tutte le risorse tradotte, semplificando le modifiche o l'aggiunta di traduzioni. In più, è facile trovare le traduzioni mancanti senza aprire ogni versione del file strings.xml. Puoi persino caricare il tuo file di stringhe per ordinare i servizi di traduzione.

Per iniziare, fai clic con il tasto destro del mouse su una copia del file strings.xml, poi fai clic su Apri editor delle traduzioni.

Scopri di più sull'editor delle traduzioni.
