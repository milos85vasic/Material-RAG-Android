Health Connect: ลดความซับซ้อนในการเชื่อมต่อระหว่างแอป

แอปสุขภาพและการออกกำลังกายบันทึกข้อมูลที่มีค่า เมื่อได้รับสิทธิ์จากผู้ใช้ คุณจะสามารถปลดล็อกศักยภาพสูงสุดของข้อมูลนี้ได้โดยการรวมข้อมูลเข้าด้วยกันเพื่อสร้างข้อมูลเชิงลึกที่มีประสิทธิภาพ
เริ่มต้นใช้งาน Health Connect ย้ายข้อมูลจาก Google Fit

ประโยชน์สำคัญ

ลดความซับซ้อนของ API

เพิ่มประสิทธิภาพโดยใช้ API ชุดเดียวเพื่อเข้าถึงข้อมูลสุขภาพและการออกกำลังกายของผู้ใช้
ดูภาพรวม

สคีมาข้อมูลแบบมาตรฐาน

อ่าน เขียน และรวบรวมข้อมูลจากแอปและอุปกรณ์ต่างๆ
อ่านเอกสารประกอบเกี่ยวกับ API

การควบคุมความเป็นส่วนตัวแบบรวมศูนย์

สร้างประสบการณ์ที่ดีขึ้นสำหรับผู้ใช้หลายพันล้านคน ทำให้ผู้ใช้สามารถควบคุมและจัดการการเข้าถึงข้อมูลของตนได้อย่างง่ายดายในระดับรายละเอียด
ดูหลักเกณฑ์ UX

ดูวิธีที่พาร์ทเนอร์ของเราผสานรวม Health Connect

Peloton, ŌURA และ Lifesum ช่วยให้ผู้ใช้เชื่อมต่อข้อมูลสุขภาวะของตนในแอปต่างๆ ได้

Health Connect ช่วยให้ ŌURA เพิ่มจำนวนการออกกำลังกายที่นำเข้าเป็น 2 เท่า, Lifesum ลดเวลาในการพัฒนาลง 75% และ Peloton มีสมาชิกเพิ่มขึ้น 10% จากการใช้แอปเพื่อบันทึกการออกกำลังกาย
อ่านเพิ่มเติม

Withings ลดการใช้โค้ดซิงค์ข้อมูลลง 50% โดยการปรับปรุงการผสานรวม API ของ Health

Withings ได้ผสานรวม Health Connect เพื่อขยายการเข้าถึงของเมตริกเพิ่มเติมและเปิดโอกาสให้ผู้ใช้ Withings ได้ใช้แอปของตนกับแอปและอุปกรณ์ที่ไม่ได้ใช้ Withings
อ่านเพิ่มเติม

Signos ปรับปรุงการผสานรวมกับแอปสุขภาพและการออกกำลังกายอื่นๆ

การใช้ Health Connect API เดียวทำให้ Signos สามารถผสานรวมข้อมูลเชิงลึกด้านสุขภาวะเพิ่มเติมในแอป และรวมข้อมูลสุขภาพและการออกกำลังกายจากแอปที่สมาชิกใช้มากที่สุด

API ข้อมูลเมตาและการเปลี่ยนแปลงใหม่ช่วยมอบประสบการณ์ที่ดียิ่งขึ้นซึ่งจะช่วยพัฒนาความสอดคล้องของข้อมูลย้อนหลัง"

ข่าวสารและวิดีโอ

สุขภาพที่เปลี่ยนแปลงไปใน Android: การย้ายข้อมูลจาก Google Fit API ไปยัง Android Health

เรายินดีที่จะประกาศให้ทราบว่าบริการสำหรับนักพัฒนาซอฟต์แวร์ Google Fit กำลังจะเปลี่ยนไปกลายเป็นส่วนหลักของแพลตฟอร์ม Android Health การเปลี่ยนแปลงนี้ช่วยให้เราให้บริการนักพัฒนาซอฟต์แวร์และผู้ใช้ได้ดียิ่งขึ้น โดยมอบประสบการณ์การใช้งานที่ราบรื่นทั่วทั้งแพลตฟอร์ม
ดูข้อมูลเพิ่มเติม

วิดีโอ: การสร้างประสบการณ์ที่ปรับเปลี่ยนได้ด้วย Android Health

รับข่าววงในเกี่ยวกับการอัปเดต Health Connect และบริการด้านสุขภาพ ดูเครื่องมือที่มีให้เลือกเพื่อสร้างรากฐานที่แข็งแกร่งและขับเคลื่อนด้วยข้อมูลสำหรับแอปที่ขับเคลื่อนโดย Android Health ของคุณ รวมถึงประเภทข้อมูลและเครื่องมือทดสอบใหม่ๆ
ดู

ยกระดับแอปของคุณด้วยฟีเจอร์ล่าสุดจาก Android Health
