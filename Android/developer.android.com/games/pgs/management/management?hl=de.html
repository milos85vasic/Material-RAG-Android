Play Game Services Management API – Übersicht
Mit Sammlungen den Überblick behalten Sie können Inhalte basierend auf Ihren Einstellungen speichern und kategorisieren.

Mit der Management API können Sie REST-Aufrufe ausgeben, um die Metadaten, die den Google Play Games Services-Funktionen zugrunde liegen, programmatisch zu steuern. Diese API vereinfacht das Testen von Play-Spieldienste-Funktionen und bietet Ihnen Flexibilität bei der Verwaltung Ihrer Spiele, um Betrug zu bekämpfen und Spieler-Konten zu korrigieren.

Mit der Management API können Sie folgende Aufgaben ausführen:

  • Setzen Sie die Werte für diese Ressourcen zurück:
    • Erfolge
    • Termine
    • Rundenbasiertes Spiel (veraltet)
    • Echtzeit-Chatroom (eingestellt)
    • Metadaten für Bestenlisten für den aktuell authentifizierten Spieler oder für alle Spieler
  • Spieler in Bestenlisten in veröffentlichten Spielen ausblenden

Weitere Informationen finden Sie in der API-Referenz und in der Beispiel-App für Verwaltungstools.

Alle Inhalte und Codebeispiele auf dieser Seite unterliegen den Lizenzen wie im Abschnitt Inhaltslizenz beschrieben. Java und OpenJDK sind Marken oder eingetragene Marken von Oracle und/oder seinen Tochtergesellschaften.
