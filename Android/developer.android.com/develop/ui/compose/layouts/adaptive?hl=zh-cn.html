自适应布局简介
使用集合让一切井井有条 根据您的偏好保存内容并对其进行分类。

Android 应用可在各种设备上运行，从可折叠翻盖手机到壁挂式电视。为了在所有类型的设备上提供出色的用户体验，请让应用的界面适应不同的显示屏尺寸和配置。优秀的 Android 应用会充分利用所占用的屏幕空间，并在运行时处理对该空间的更改，包括分屏和桌面窗口模式下的屏幕方向更改和窗口大小调整。

注意 ：对于以 Android 16（API 级别 36）为目标平台的应用，系统会忽略屏幕方向、宽高比和应用可调整大小限制，以改善最小宽度为 600dp 或更大的设备规格上的应用布局。请参阅应用屏幕方向、宽高比和可调整大小。

主题

自适应布局文档提供了以下方面的指导：

  • 设计和实现自适应布局
  • 根据窗口大小调整应用的主要导航栏
  • 使用窗口大小类别调整应用的界面
  • 使用 Jetpack API 简化规范布局（例如列表-详情）的实现
