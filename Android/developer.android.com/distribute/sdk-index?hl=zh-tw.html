Google Play SDK 索引

Google Play SDK 索引提供了 Google Play 上熱門商用 SDK 的深入分析和使用資料，讓您能夠瞭解特定 SDK 是否適合您的業務和使用者，據此做出明智決策。

您可以在這裡找到最廣泛運用的 100 多個商用 SDK，以及個別 SDK 的深入分析。

前往 Google Play SDK 索引

查看重要深入分析

在將 SDK 加入應用程式前，您可以查看 SDK 的可靠性和安全性信號，瞭解版本採用率、續用率及 SDK 要求的應用程式權限。
搜尋 SDK

查看資料安全性指南

取得 SDK 指南的連結，瞭解 SDK 收集的資料類型與收集原因，以便填寫應用程式的資料安全性表單。
資料安全性表單

認識承諾遵守 Google Play 政策的合作夥伴

看看哪些 SDK 供應商已聲明會避免在自家 SDK 中使用可能導致應用程式違反 Google Play 政策的程式碼。
瞭解詳情

SDK 供應商與 Google Play 合作的方式

SDK 供應商可以在 Google Play SDK 管理中心聲明自己的 SDK 擁有權並註冊 SDK，以便查看統計資料和當機報告，以及與開發人員分享資訊。這些供應商也可以透過 Google Play 管理中心和 Android Studio 向您傳送緊急訊息。萬一您的應用程式版本使用了有問題的 SDK 版本，Google Play 可以在該版應用程式發布前向您發出警告。
