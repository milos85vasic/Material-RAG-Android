Crie infinitas possibilidades com o Android XR

Crie experiências que ultrapassam os limites e transformam a forma como as pessoas criam, exploram e se divertem, com o Android XR. Os recursos imersivos e as ferramentas confiáveis da plataforma significam que o único limite é sua imaginação. Comece hoje mesmo desenvolvendo para headsets, com mais novidades para o desenvolvimento de óculos no futuro.
Desenvolver seu app

Escolha o framework de sua preferência: Android Jetpack XR, Unity, OpenXR ou WebXR.

Crie apps para a próxima geração de computação usando ferramentas de desenvolvimento do Android, o mecanismo de jogo Unity, os padrões abertos do OpenXR ou o WebXR amplamente acessível.

Android Jetpack XR

A maioria dos apps Android é compatível com o Android XR sem nenhum esforço extra de desenvolvimento. O Android XR processa a interação com o dispositivo e o gerenciamento do ciclo de vida para que você possa se concentrar na inovação.
Desenvolver com o SDK Jetpack XR

Unity

Acesse todos os recursos de produção de conteúdo do Unity e leve apps de outras plataformas para o Android XR com facilidade. Garanta um desenvolvimento tranquilo com ferramentas de otimização de desempenho, uma grande loja de recursos e uma comunidade forte.
Desenvolver com o Unity

OpenXR

Simplifique o desenvolvimento com o padrão aberto sem royalties do OpenXR. Crie em qualquer lugar usando um conjunto comum de APIs para desenvolver apps de RA que funcionam em vários dispositivos.
Desenvolver com o OpenXR

WebXR

Use o poder das tecnologias da Web para criar experiências de RA diretamente em um navegador. O WebXR disponibiliza a realidade estendida para qualquer pessoa com um dispositivo e um navegador da Web compatível.
Desenvolver com o WebXR

Começar a usar o Android XR

Projetar para Android XR

As diretrizes de design do Android XR incluem sistemas de design prontos para uso, estilos, dimensionamento de interface, componentes, painéis espaciais, modelos 3D e ambientes.

Seguir os princípios de XR

Comece de onde você está, faça os usuários se sentirem confortáveis, explore experiências que pareçam especiais em XR e garanta que seu app seja acessível.

Aprenda os fundamentos da XR

O Android XR ajuda você a criar apps de XR de alta qualidade rapidamente. É possível espacializar o conteúdo, os layouts e os elementos da interface de um app usando componentes pré-criados.

Usar interações naturais

O Android XR oferece aos usuários flexibilidade e controle. Os usuários podem interagir com seu app usando rastreamento de mãos e olhos, comandos de voz e dispositivos Bluetooth.

Simplifique a criação

Leve apps 2D para a RA com facilidade ou aproveite ao máximo as experiências 3D. Acelere o desenvolvimento com ferramentas confiáveis e APIs líderes do setor.
