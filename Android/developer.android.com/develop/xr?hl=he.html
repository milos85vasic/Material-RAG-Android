אפליקציה עם תכונות ייחודיות ל-XR

כדי ליצור חוויה עשירה, כדאי להשתמש ב חלוניות מרחבי, במודלים תלת-ממדיים ובסביבות מרחביות. לחלופין, אפשר להשתמש ב-Unity, ב-OpenXR או ב-WebXR כדי ליצור חוויה מרתקת בהתאמה אישית מלאה.

כלי פיתוח ל-Android

יצירת חוויות XR באמצעות כלים מוכרים שהורחבו ל-XR.

Android Studio

יצירת תהליך מכשיר חדש, מנהל מכשירי חדש ובדק פריסה חדש. אתם יכולים להשתמש באמולטור Android XR כדי לבדוק אפליקציות ישירות במחשב, עם או בלי גישה למכשיר פיזי.

Jetpack פיתוח נייטיב ל-XR

כולל את כל מה שצריך כדי לעצב אפליקציית XR. כאן תוכלו למצוא המלצות לממשק משתמש ולאינטראקציה, מנוע פריסה אוטומטי, פריסות מוגדרות מראש וקומפוזיציות של ממשק משתמש מרחבי.

ARCore ל-Jetpack XR

הוספת יכולות תפיסת נתונים לאפליקציה כדי שהיא תבין ותתקשר עם תכונות בעולם האמיתי, כמו עוגנים ופיצול סמנטי.

Jetpack SceneCore

הספרייה כוללת את אבני הבניין להוספת תוכן immersive לאפליקציית Android XR. ספריית SceneCore תומכת בממשקי API של ישויות מרחביות וסביבות.

Material Design ל-XR

מספקת בסיס ליצירת אפליקציות ידידותיות למשתמש, שתואמות לדפוסים מקובלים של אינטראקציה, עיצוב מרחבי ועיצוב חזותי.

כלים תלת-ממדיים

משתמשים בכלי התלת-ממד שבוחרים כדי ליצור קובצי glTF או GLB. דוגמאות: Blender, ‏ Maya, ‏ Spline

אפליקציית Hello Android XR לדוגמה

כדאי לנסות את Hello Android XR – אפליקציה לדוגמה, שמותאמת לחוויה immersive. איך פאנלים וסביבות מרחביות משתלבים ביחד ביישום בעולם האמיתי.

הדרכות Codelab ל-Android XR

בעזרת המדריכים המפורטים תוכלו ללמוד על אבני הבניין של חוויות Android XR ולהתאים אפליקציה קיימת כך שתוכל להשתמש בהן.
הדרכה ראשונה ב-Codelab

פיתוח באמצעות Unity,‏ OpenXR או WebXR

כל מי שמכיר פיתוח תלת-ממד יכול ליצור באמצעות מנוע התלת-ממד הפופולרי בזמן אמת של Unity, תקן התעשייה OpenXR או WebXR שזמין בקלות. המסגרות האלה מספקות לכם את הגמישות הנדרשת כדי ליצור את חוויות ה-XR שאתם רוצים.

Unity

פיתוח אפליקציות שיכולות לטרגט כמה פלטפורמות בעזרת תמיכת OpenXR המוכחת של Unity וסביבת הפיתוח שלה.
