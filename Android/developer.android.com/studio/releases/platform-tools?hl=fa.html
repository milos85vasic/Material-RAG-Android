یادداشت‌های انتشار SDK Platform Tools
با مجموعه‌ها، منظم بمانید ذخیره و طبقه‌بندی محتوا براساس اولویت‌های شما.

Android SDK Platform-Tools یک جزء برای Android SDK است. این شامل ابزارهایی است که با پلتفرم اندروید، در درجه اول adb و fastboot ، رابط دارند. اگرچه adb برای توسعه برنامه اندروید مورد نیاز است، توسعه دهندگان برنامه معمولاً فقط از نسخه نصب استودیو استفاده می کنند. اگر می خواهید از adb مستقیماً از خط فرمان استفاده کنید و Studio را نصب نکرده باشید، این دانلود مفید است. (اگر استودیو را نصب کرده اید، ممکن است بخواهید فقط از کپی نصب شده استفاده کنید زیرا استودیو به طور خودکار آن را به روز می fastboot ). این بسته قبلاً حاوی systrace بود، اما به نفع Studio Profiler، gpuinspector.dev یا Perfetto منسوخ شده است.

اگرچه برخی از ویژگی‌های جدید در adb و fastboot فقط برای نسخه‌های اخیر اندروید در دسترس هستند، اما با نسخه‌های قبلی سازگار هستند، بنابراین شما فقط باید به آخرین نسخه SDK Platform-Tools نیاز داشته باشید و در صورت یافتن استثنا باید باگ‌ها را ثبت کنید.

دانلودها
