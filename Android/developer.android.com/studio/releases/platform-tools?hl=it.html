Note di rilascio di SDK Platform Tools
Mantieni tutto organizzato con le raccolte Salva e classifica i contenuti in base alle tue preferenze.

Android SDK Platform-Tools è un componente dell'SDK Android. Include strumenti che interagiscono con la piattaforma Android, principalmente adb e fastboot. Sebbene adb sia necessario per lo sviluppo di app per Android, gli sviluppatori di app normalmente utilizzano solo la copia di Studio installata. Questo download è utile se vuoi utilizzare adb direttamente dalla riga di comando e non hai installato Studio. Se hai installato Studio, ti consigliamo di utilizzare la copia installata, perché Studio la aggiornerà automaticamente. fastboot è necessario se vuoi sbloccare il bootloader del dispositivo e installare una nuova immagine di sistema. Questo pacchetto conteneva systrace, ma è stato ritirato a favore di Studio Profiler, gpuinspector.dev o Perfetto.

Sebbene alcune nuove funzionalità di adb e fastboot siano disponibili solo per le versioni recenti di Android, sono retrocompatibili, quindi dovresti aver bisogno solo dell'ultima versione di SDK Platform-Tools e segnalare i bug se trovi eccezioni.

I download

Se sei uno sviluppatore Android, devi scaricare l'ultima versione di SDK Platform-Tools da SDK Manager di Android Studio o dallo strumento da riga di comando sdkmanager. In questo modo, gli strumenti vengono salvati nella posizione corretta insieme al resto degli strumenti dell'SDK Android e possono essere aggiornati facilmente.

  • adb
    • Backend USB libusb riscritto (utilizza l'API di sincronizzazione anziché l'API asincrona). Migliora l'affidabilità e corregge l'esaurimento della memoria su Linux.

    • È stato risolto il problema di utilizzo dopo la liberazione della memoria nel backend libusb.

  • adb

    • Rendi il backend mDNS openscreen predefinito su tutte le piattaforme.
    • Supporto per il rilevamento di USB SuperSpeed+ (velocità attuali e negoziate) a scopo diagnostico.
    • Arresto normale: rilascia tutte le interfacce USB all'arresto (tutti i sistemi operativi).

  • fastboot
    • È stato corretto flashall sui Pixel 3.
