開始使用 Android 11

  1. 設定執行階段環境：請參閱「取得 Android 11」一文，瞭解如何刷新 Google Pixel 裝置或設定模擬器。
  2. 設定 Android Studio：試用 Android 11 SDK 和工具。如需操作步驟，請參閱設定指南。
  3. 瞭解最新功能：查看可能影響應用程式的隱私權功能和行為變更。
  4. 測試應用程式：逐步完成所有流程，找出問題。在執行階段切換行為變更，以便找出問題。
  5. 更新應用程式：盡可能指定 Android 11，並透過 Beta 版管道或其他群組與使用者進行測試。
