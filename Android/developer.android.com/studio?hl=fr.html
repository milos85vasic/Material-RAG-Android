Android Studio

L'IDE officiel pour le développement d'applications Android accélère désormais votre productivité grâce à Gemini dans Android Studio, votre assistant de codage basé sur l'IA.

Votre appareil actuel n'est pas compatible.
Consultez la configuration système requise.

Consulter les notes de version

Nouvelles fonctionnalités

Caractéristique

Essayez Gemini dans Android Studio

Gemini dans Android Studio est un assistant IA qui vous aide à générer et à corriger du code, et à répondre à vos questions sur le développement d'applications Android.

Essayez Gemini dans Android Studio
Caractéristique

Coder n'importe où avec Android Studio Cloud

Essayez Android Studio Cloud directement dans votre navigateur, accessible via Firebase Studio. Simplifiez votre workflow et évitez les installations locales. Essayez la version Preview en accès anticipé.

Essayer Android Studio Cloud

Fonctionnalités essentielles

Design

Outils de conception Compose

Créez des mises en page dynamiques avec Jetpack Compose. Ensuite, prévisualisez vos mises en page sur n'importe quelle taille d'écran et inspectez les animations Compose à l'aide des outils d'inspection intégrés.
Concevoir avec Compose
Développer

Éditeur de code intelligent

Rédigez du code de meilleure qualité, travaillez plus rapidement et gagnez en productivité grâce à un éditeur de code intelligent qui fournit la saisie semi-automatique pour les langages de programmation Kotlin, Java et C/C++. De plus, lorsque vous modifiez Jetpack Compose, vous pouvez voir immédiatement les modifications apportées à votre code grâce à la modification en temps réel.
En savoir plus sur l'éditeur
Créer

Système de compilation flexible

Basé sur Gradle, le système de compilation d'Android Studio vous permet de personnaliser votre compilation pour générer plusieurs variantes de compilation pour différents appareils Android à partir d'un seul projet. Analysez ensuite les performances de vos builds et identifiez les problèmes potentiels dans votre projet à l'aide de Build Analyzer.

En savoir plus sur Android Build
Test

Émulez facilement n'importe quel appareil

Android Emulator vous permet de tester votre application sur divers appareils Android. Exploitez tout le potentiel de vos applications grâce à des mises en page responsives qui s'adaptent aux téléphones, tablettes, appareils pliables, Wear OS, téléviseurs et appareils ChromeOS.

Utiliser Android Emulator

Android App Bundle

Trouvez des opportunités d'optimiser la taille de votre application Android avant sa publication en inspectant le contenu du fichier APK ou de l'Android App Bundle de votre application. Inspectez le fichier manifeste, les ressources et les fichiers DEX. Comparez deux APK ou fichiers Android App Bundle pour connaître l'évolution de la taille de votre application d'une version à l'autre.
