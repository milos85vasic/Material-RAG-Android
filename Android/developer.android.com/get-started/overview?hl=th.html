เริ่มต้นใช้งาน Android

หากต้องการเริ่มต้นใช้งาน ให้ทำ Codelab สั้นๆ หรือเจาะลึกรายละเอียดหลักสูตรการฝึกอบรมเพื่อดูภาพรวมของหัวข้อการพัฒนา Android ที่สำคัญ

สร้างแอป Hello World

ทำตาม Codelab แนะนำนี้พร้อมวิธีการทีละขั้นตอนในการสร้างแอป Hello World แบบเรียบง่าย
เริ่มใช้ Codelab

สร้างแอป Android ที่ปรับขนาดได้ เป็นแบบโมดูล และทดสอบได้

บทแนะนำนี้จะแสดงวิธีสร้างแอปที่ทดสอบได้ เป็นแบบโมดูล และปรับขนาดได้ตั้งแต่เริ่มต้น แอปนี้จะไม่เป็นแอป Hello World เราจะมุ่งเน้นที่การสร้างสถาปัตยกรรมแอปที่จะช่วยให้แอปเติบโตได้
หลักสูตร

เริ่มหลักสูตรข้อมูลพื้นฐานเกี่ยวกับ Android

เข้าร่วมหลักสูตรเต็มเพื่อเรียนรู้พื้นฐานของการสร้างแอปด้วย Jetpack Compose, ชุดเครื่องมือสมัยใหม่ของ Android สำหรับการพัฒนาอินเทอร์เฟซผู้ใช้

เมื่อสร้างชุดแอป คุณจะได้เรียนรู้พื้นฐานเกี่ยวกับการเขียนโปรแกรม Kotlin ภาษาและพื้นฐานการพัฒนาแอป

เรียนรู้

เรียกดูแหล่งข้อมูลการเรียนรู้

เจาะลึกยิ่งขึ้นด้วยการสำรวจแหล่งข้อมูลการฝึกอบรมอื่นๆ เช่น เส้นทางการเรียนรู้สำหรับ หัวข้อที่ซับซ้อนขึ้น รวมถึงการเขียน สถาปัตยกรรมแอป และการช่วยเหลือพิเศษ

คุณภาพของแอปเป็นสิ่งสำคัญ

หากคุณเรียนรู้ได้ดีที่สุดจากการอ่านโค้ด เรามีตัวอย่างแอปมากมายให้คุณได้ดู แก้ไข และเรียนรู้

Now in Android

ตอนนี้ Android เป็นแอป Android ที่สร้างด้วย Kotlin และ Jetpack Compose ตอนนี้ Android ออกแบบมาเพื่อทำหน้าที่เป็นข้อมูลอ้างอิงสำหรับนักพัฒนาซอฟต์แวร์ Android ซึ่งเป็นต้นแบบของแนวทางปฏิบัติแนะนำสำหรับการออกแบบและการพัฒนา

เขียนตัวอย่าง

ที่เก็บนี้ประกอบด้วยชุดโปรเจ็กต์ Android Studio แต่ละโปรเจ็กต์ที่จะช่วยให้คุณเรียนรู้เกี่ยวกับ Compose ใน Android ตัวอย่างแต่ละรายการจะแสดงให้เห็นกรณีการใช้งาน ระดับความซับซ้อน และ API ที่ต่างกัน

แอปสิ่งที่ต้องทำ

ตัวอย่างเหล่านี้แสดงแนวทางเชิงสถาปัตยกรรมที่แตกต่างกันในการพัฒนาแอป Android ในสาขาที่ต่างกัน คุณจะเห็นแอปเดียวกัน (แอป TODO) มีการใช้งานที่แตกต่างกันเล็กน้อย
