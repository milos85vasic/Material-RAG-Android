SDK বিল্ড টুল রিলিজ নোট
সেভ করা পৃষ্ঠা গুছিয়ে রাখতে 'সংগ্রহ' ব্যবহার করুন আপনার পছন্দ অনুযায়ী কন্টেন্ট সেভ করুন ও সঠিক বিভাগে রাখুন।

Android SDK Build-Tools হল Android SDK-এর একটি উপাদান যা Android অ্যাপ তৈরির জন্য প্রয়োজনীয়৷ এটি <sdk>/build-tools/ ডিরেক্টরিতে ইনস্টল করা আছে।

অ্যান্ড্রয়েড SDK ম্যানেজার ব্যবহার করে সর্বশেষ সংস্করণ ডাউনলোড করে আপনার বিল্ড টুলস কম্পোনেন্ট সবসময় আপডেট রাখা উচিত। আপনি যদি Gradle 3.0.0 বা উচ্চতর জন্য Android প্লাগইন ব্যবহার করেন, তাহলে আপনার প্রকল্প স্বয়ংক্রিয়ভাবে প্লাগইন নির্দিষ্ট করা বিল্ড টুলের একটি ডিফল্ট সংস্করণ ব্যবহার করে। বিল্ড টুলের একটি ভিন্ন সংস্করণ ব্যবহার করতে, আপনার মডিউলের build.gradle এ buildToolsVersion ব্যবহার করে এটি উল্লেখ করুন, নিম্নরূপ:

গ্রোভি 

android {
    buildToolsVersion "35.0.0"
    ...
}

কোটলিন 

android {
    buildToolsVersion = "35.0.0"
    ...
}

রিভিশন

নীচের বিভাগগুলি বিল্ড টুলের রিলিজ সম্পর্কে নোট প্রদান করে। আপনার SDK-এ বিল্ড টুলগুলির কোন সংশোধনগুলি উপলব্ধ তা নির্ধারণ করতে, Android SDK ম্যানেজারে ইনস্টল করা প্যাকেজ তালিকা দেখুন৷
