開始使用 Android 14

  1. 設定執行階段環境：請參閱取得 Android 14，瞭解如何刷新 Google Pixel 裝置或設定模擬器。
  2. 設定 Android Studio：試用 Android 14 SDK 和工具。相關步驟請參閱 SDK 設定頁面。
  3. 瞭解新功能：查看所有應用程式的行為變更，以及以 Android 14 為目標版本可能影響應用程式的行為變更。
  4. 測試應用程式 - 執行所有流程，檢查是否有問題。 在執行階段切換行為變更，藉此找出問題。
  5. 更新應用程式 - 盡可能指定 Android 14，並使用 Beta 版或其他群組對使用者進行測試。
