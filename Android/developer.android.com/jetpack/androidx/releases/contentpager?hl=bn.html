নির্ভরতা ঘোষণা করা

ContentPager-এ নির্ভরতা যোগ করতে, আপনাকে অবশ্যই আপনার প্রকল্পে Google Maven সংগ্রহস্থল যোগ করতে হবে। আরও তথ্যের জন্য Google এর Maven সংগ্রহস্থল পড়ুন।

আপনার অ্যাপ বা মডিউলের জন্য build.gradle ফাইলে আপনার প্রয়োজনীয় আর্টিফ্যাক্টগুলির জন্য নির্ভরতা যোগ করুন:

গ্রোভি 

dependencies {
    implementation "androidx.contentpager:contentpager:1.0.0"
}

কোটলিন 

dependencies {
    implementation("androidx.contentpager:contentpager:1.0.0")
}
