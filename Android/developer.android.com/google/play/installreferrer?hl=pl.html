Strona odsyłająca do instalacji z Google Play
Zadbaj o dobrą organizację dzięki kolekcji Zapisuj i kategoryzuj treści zgodnie ze swoimi preferencjami.

Do bezpiecznego pobierania z Google Play treści odesłań możesz używać interfejsu instalacji Referrer API w Sklepie Google Play, na przykład:

  • Adres URL strony odsyłającej zainstalowanego pakietu.
  • Sygnatura czasowa (w sekundach) kliknięcia strony odsyłającej (zarówno po stronie klienta, jak i po stronie serwera).
  • Sygnatura czasowa (w sekundach) rozpoczęcia instalacji (po stronie klienta i po stronie serwera).
  • Wersja aplikacji w momencie jej pierwszej instalacji.
  • Określa, czy w ciągu ostatnich 7 dni użytkownik korzystał z aplikacji błyskawicznej w Twojej aplikacji.

Wymagania

Interfejs Admin Referrer API jest widoczny w aplikacji Sklep Google Play na urządzeniu. Urządzenia z aplikacją Google Play w wersji 8.3.73 lub nowszej automatycznie mają dostęp do interfejsu API.

Aby korzystać z interfejsu Admin Referrer API, musisz mieć konto w Konsoli Google Play.

Korzystanie z interfejsu API

Interfejs Admin Referrer API jest zaimplementowany jako interfejs Android Interface Definition Language (AIDL).

  • Jeśli używasz języka programowania Kotlin lub języka programowania Java, skorzystaj z biblioteki klienta odsyłającego do instalacji z Play, aby uprościć kodowanie.
  • Jeśli używasz innego języka programowania, użyj interfejsu PlayInstall Referrer API.
