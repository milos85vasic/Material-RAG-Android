แอปอื่นๆ ที่เกี่ยวข้องกับการขับรถ
จัดทุกอย่างให้เป็นระเบียบอยู่เสมอด้วยคอลเล็กชัน บันทึกและจัดหมวดหมู่เนื้อหาตามค่ากำหนดของคุณ

Android for Cars มีเทมเพลตที่ช่วยให้คุณสร้างประสบการณ์การใช้งานแอปที่หลากหลายเพื่อช่วยผู้ขับขี่ในการเดินทาง ตั้งแต่การค้นหาตัวเลือกที่จอดรถและสถานีชาร์จ ไปจนถึงการตรวจสอบสภาพอากาศและการสื่อสารกับอุปกรณ์ในบ้านขณะอยู่บนท้องถนน

ไลบรารีแอป Android สำหรับรถยนต์มีเทมเพลตที่ปรับให้เหมาะกับการขับขี่ (บางเทมเพลตมี แผนที่ที่วาดโดยไลบรารีแอป) รวมถึงเทมเพลตที่มีตัวแปรที่จอดอยู่หรือ เน้นเฉพาะการจอดรถ การรวมเทมเพลตเหล่านี้ในโฟลว์งานจะช่วยให้คุณออกแบบประสบการณ์การใช้งานที่ใช้ได้ทั้ง Android Auto และ AAOS ในสถานการณ์ในรถยนต์ที่หลากหลาย

ปัจจุบันหมวดหมู่แอปที่ได้รับอนุมัติสำหรับไลบรารีแอป Android สำหรับรถยนต์ (นอกเหนือจากหมวดหมู่การนำทางที่กล่าวถึงในแอปนำทาง) ได้แก่ จุดที่น่าสนใจ (เช่น ที่จอดรถและที่ชาร์จ) และ IoT

ดูข้อมูลเพิ่มเติมเกี่ยวกับแอปสภาพอากาศได้ที่แอปสภาพอากาศ

ตัวอย่างเนื้อหาและโค้ดในหน้าเว็บนี้ขึ้นอยู่กับใบอนุญาตที่อธิบายไว้ในใบอนุญาตการใช้เนื้อหา Java และ OpenJDK เป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของ Oracle และ/หรือบริษัทในเครือ
