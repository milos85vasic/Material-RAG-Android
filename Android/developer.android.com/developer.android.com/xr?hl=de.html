Mit Android XR unendliche Möglichkeiten schaffen

Mit Android XR können Sie neue Grenzen überschreiten und die Art und Weise verändern, wie Menschen Inhalte erstellen, entdecken und genießen. Dank der immersiven Funktionen und zuverlässigen Tools der Plattform sind Ihrer Kreativität keine Grenzen gesetzt. Sie können schon heute mit der Entwicklung für Headsets beginnen. In Zukunft werden auch weitere Funktionen für die Entwicklung von intelligenten Brillen hinzukommen.
App entwickeln

Wählen Sie das gewünschte Framework aus: Android Jetpack XR, Unity, OpenXR oder WebXR.

Mit bekannten Android-Entwicklungstools, der beliebten Game Engine Unity, den offenen Standards von OpenXR oder dem weit verbreiteten WebXR können Sie Apps für die nächste Generation von Computing erstellen.

Android Jetpack XR

Die meisten Android-Apps sind ohne zusätzlichen Entwicklungsaufwand mit Android XR kompatibel. Android XR übernimmt die Geräteinteraktion und die Lebenszyklusverwaltung, sodass Sie sich auf Innovationen konzentrieren können.
Mit dem Jetpack XR SDK entwickeln

Unity

Sie erhalten vollen Zugriff auf die Funktionen zur Inhaltsproduktion von Unity und können Apps ganz einfach von anderen Plattformen auf Android XR übertragen. Mit Tools zur Leistungsoptimierung, einem großen Asset-Shop und einer starken Community sorgen Sie für eine reibungslose Entwicklung.
Mit Unity entwickeln

OpenXR

Mit dem gebührenfreien OpenXR-Standard die Entwicklung optimieren Mit einer gemeinsamen Reihe von APIs XR-Apps entwickeln, die auf einer Vielzahl von Geräten funktionieren.
Mit OpenXR entwickeln

Logo: WebXR

Mithilfe von Webtechnologien können Sie XR-Inhalte direkt in einem Browser erstellen. Mit WebXR steht erweiterte Realität allen Nutzern mit einem Gerät und einem unterstützten Webbrowser zur Verfügung.
Mit WebXR entwickeln

Erste Schritte mit Android XR

Design für Android XR

Die Android XR-Designrichtlinien umfassen fertige Designsysteme, Stile, UI-Skalierung, Komponenten, räumliche Bereiche, 3D-Modelle und Umgebungen.

XR-Grundsätze einhalten

Beginnen Sie dort, wo Sie sind, sorgen Sie dafür, dass sich die Nutzer wohlfühlen, erkunden Sie Funktionen, die in XR besonders sind, und achten Sie darauf, dass Ihre App barrierefrei ist.

XR-Grundlagen

Mit Android XR können Sie schnell hochwertige XR-Apps entwickeln. Sie können die Inhalte, Layouts und UI-Elemente einer vorhandenen App mit vordefinierten Komponenten ordnen.

Natürliche Interaktionen verwenden

Android XR bietet Nutzern Flexibilität und Kontrolle. Nutzer können über Gesten- und Augen-Tracking, Sprachbefehle und Bluetooth-Geräte mit Ihrer App interagieren.

Neue Funktionen

Android XR-Entwicklervorschau

Sie können als einer der ersten Entwickler eine Android XR-App entwickeln.
Mit Android XR entwickeln

Beispiel-App „Hello Android XR“ ausprobieren

Sie möchten wissen, was Android XR kann? Sehen Sie sich die Beispiel-App „Hello Android XR“ an. Hier können Sie sich selbst davon überzeugen, wie sich räumliche Bereiche und Umgebungen in einer realen Anwendung kombinieren lassen.
Beispiel-App ausprobieren

Barrierefreies Bauen

Mithilfe von räumlichen Frameworks, unbegrenztem Displaybereich und multimodalen Eingaben können Sie Ihre Visionen zum Leben erwecken und so für grenzenlose Kreativität und dynamische Funktionen sorgen.

Erstellung vereinfachen

Sie können ganz einfach 2D-Apps für XR-Geräte bereitstellen oder sich ganz auf 3D-Inhalte konzentrieren. Mit bewährten Tools und branchenführenden APIs eine schnelle Entwicklung ermöglichen

Mit Android für die Zukunft skalieren

Erhöhen Sie die Reichweite Ihrer App mit dem offenen Ökosystem von Geräteherstellern von Android XR und der großen Nutzerbasis des Play Store.
