Compose Material 3 Adaptive

  
Documentation de référence de l'API
androidx.compose.material3.adaptive
Utilisez la bibliothèque adaptative Material 3 pour créer des UI adaptatives qui s'adapteront automatiquement en fonction des configurations de fenêtre actuelles, comme les classes de taille de fenêtre ou les postures de l'appareil. La bibliothèque fournit à la fois des implémentations de scaffold par défaut et les composables de blocs de construction nécessaires pour créer vos propres expériences personnalisées.
Dernière mise à jour Version stable Version finale Version bêta Version alpha
27 août 2025 1.1.0 - 1.2.0-beta01 -

Déclarer des dépendances

Pour ajouter une dépendance sur Compose, vous devez ajouter le dépôt Maven de Google à votre projet. Pour en savoir plus, consultez la section Dépôt Maven de Google.

Ajoutez les dépendances correspondant aux artefacts dont vous avez besoin dans le fichier build.gradle de votre application ou de votre module :

Groovy

dependencies {
    implementation "androidx.compose.material3.adaptive:adaptive:1.2.0-beta03"
    implementation "androidx.compose.material3.adaptive:adaptive-layout:1.2.0-beta03"
    implementation "androidx.compose.material3.adaptive:adaptive-navigation:1.2.0-beta03"
}

Kotlin

dependencies {
    implementation("androidx.compose.material3.adaptive:adaptive:1.2.0-beta03")
    implementation("androidx.compose.material3.adaptive:adaptive-layout:1.2.0-beta03")
    implementation("androidx.compose.material3.adaptive:adaptive-navigation:1.2.0-beta03")
}

Pour en savoir plus sur les dépendances, consultez Ajouter des dépendances de compilation.

Commentaires

Vos commentaires nous aident à améliorer Jetpack. N'hésitez pas à nous contacter si vous découvrez de nouveaux problèmes ou si vous avez des idées pour améliorer cette bibliothèque. Veuillez consulter les problèmes existants de cette bibliothèque avant d'en signaler un nouveau. Vous pouvez ajouter votre vote à un problème existant en cliquant sur le bouton en forme d'étoile.

Signaler un nouveau problème

Pour en savoir plus, consultez la documentation sur l'outil Issue Tracker.

Publication d'androidx.compose.material3.adaptive:adaptive-navigation3:1.0.0-alpha02 et d'androidx.compose.material3.adaptive:adaptive-navigation3-android:1.0.0-alpha02. La version 1.0.0-alpha02 contient ces commits.

Version 1.0.0-alpha01

27 août 2025

Publication d'androidx.compose.material3.adaptive:adaptive-navigation3:1.0.0-alpha01 et d'androidx.compose.material3.adaptive:adaptive-navigation3-android:1.0.0-alpha01. La version 1.0.0-alpha01 contient ces commits.

Nouvelles fonctionnalités

Nous avons le plaisir de vous annoncer la première version alpha de la bibliothèque adaptive-navigation3. Cette nouvelle bibliothèque, qui fait partie de l'écosystème Material Adaptive et Navigation3, est conçue pour vous aider à créer des UI Material adaptatives intégrées à la bibliothèque Navigation3.

Utilisez des stratégies de scène pour implémenter un scaffold de volet liste/détails canonique (ListDetailSceneStrategy, rememberListDetailSceneStrategy) ou un scaffold de volet d'assistance (SupportingPaneSceneStrategy, rememberSupportingPaneSceneStrategy). Pour en savoir plus sur les scènes, les stratégies de scène et les autres nouveaux concepts introduits dans Navigation3, consultez le guide Navigation3.

Principales fonctionnalités de la version 1.0.0

  • Fonctions composables currentWindowSize(), collectFoldingFeaturesAsState() et currentWindowAdaptiveInfo() pour obtenir les informations de fenêtre requises afin d'adapter les applications, comme la taille des fenêtres et les fonctionnalités de pliage.

  • Animations Material 3 intégrées par défaut lors du changement d'état du scaffold adaptatif.
