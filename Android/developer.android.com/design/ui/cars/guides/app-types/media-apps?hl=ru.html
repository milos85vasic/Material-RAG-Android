Медиа-приложения
Оптимизируйте свои подборки Сохраняйте и классифицируйте контент в соответствии со своими настройками.

Помогите пользователям легко находить любимые песни, подкасты и аудиокниги, не отрывая взгляда от дороги. Вы можете разработать комплексный медиа-опыт, который позволит использовать все возможности вашего приложения в автомобиле.

Оптимизировано для вождения

Безопасность — ключ к успеху любого приложения, которое будет использоваться во время вождения. Мультимедийные возможности автомобиля оптимизированы для минимизации отвлекающих факторов, основываясь на следующих основных принципах:

  • Голосовое управление: позвольте водителям выполнять задачи с помощью голоса, а не нажатия кнопок.
  • Краткая последовательность задач: по возможности сократите последовательность задач до 3 шагов или меньше.
  • Только аудио во время вождения: видео может отвлекать водителей , поэтому медиаприложения ограничиваются в первую очередь аудиоконтентом.

Используйте шаблоны, чтобы максимально эффективно использовать возможности вашего приложения в автомобиле

Медиа-шаблоны предоставляют средства, помогающие вам настраивать ключевые компоненты вашего приложения, полагаясь на опыт Google в автомобильной отрасли в вопросах разработки основ и обеспечения безопасности.

Используя шаблоны для создания приложения, вы можете настроить структуру браузера в соответствии со своими потребностями. Дополнительные шаги для шаблонных приложений включают создание процесса входа или настроек экрана автомобиля (для AAOS) с использованием шаблонов из библиотеки приложений Android for Cars.

Более подробную информацию см. в разделе Создание шаблонного медиа-приложения .
