テンプレートの概要
コレクションでコンテンツを整理 必要に応じて、コンテンツの保存と分類を行います。

自動車向け Android アプリ ライブラリは、Android Auto と Android Automotive OS の両方で動作するアプリの作成に役立ちます。

自動車向け Android アプリ ライブラリ:

  • アプリのユーザー インターフェースを設計するためにタスクフローに組み立てることができる、自動車向けに最適化されたテンプレートとコンポーネントを提供します。
  • レスポンシブな画面サイズ調整などの重要な機能を処理し、アプリがすべての対応車両で効果的に動作するようにします。
  • 複数のアプリ カテゴリで利用可能: こちらで全リストをご覧ください。

このセクションでは、利用可能なすべてのテンプレートと、各テンプレートに含まれるコンポーネントについて説明します。アプリが満たすべき要件の一覧については、テンプレート アプリの要件をご覧ください。

注: テンプレートの設計プロセスは Android Auto と Android Automotive OS（AAOS）の両方で同じですが、開発プロセスでは 2 つの異なる APK を作成する必要があります。テンプレートを使用してアプリを作成するための技術的なガイダンスについては、自動車向け Android アプリ ライブラリを使用するをご覧ください。

自動車向け Android アプリ ライブラリには、次のものが含まれています。

汎用テンプレート（自動車向けアプリ ライブラリを使用して作成されたすべてのアプリタイプで使用可能）

テンプレート 表示される内容
リスト、グリッド リストまたはグリッド レイアウトの情報アイテム
セクションに区切られたアイテム リストとグリッドを使用してブラウジング構造を組み合わせる
ペイン 詳細な情報と目立つアクション（画像は省略可）
Message、Long Message 短いメッセージまたは長いメッセージの全画面表示と関連するアクション
検索 検索バーと結果リスト
ログイン アプリにログインするためのオプション
タブ 上部のタブで、他の埋め込みテンプレートのビューを切り替えます。

特別な目的のテンプレート

テンプレート 表示される内容 アプリのカテゴリ
ナビゲーション アクション ストリップとオプションのルート案内カードが表示された全画面表示の地図 ナビゲーション
場所リスト（地図） アプリ ライブラリによって描画された地図の横にある場所やその他のアイテムのリスト POI
地図 + コンテンツ 他のテンプレートのビューを含むマップ ナビゲーション、POI、天気
メディアの再生 現在再生中のメディアのビュー メディア
