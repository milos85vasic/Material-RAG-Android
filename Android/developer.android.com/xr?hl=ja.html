Android XR で無限の可能性を創造する

Android XR を活用して、エンターテイメントの創造、探索、楽しみ方を変革し、限界を押し広げるエクスペリエンスを構築しましょう。没入感のある機能と信頼できるツールを備えたこのプラットフォームでは、想像力次第で無限の可能性を広げることができます。ヘッドセット向けの開発を今すぐ開始し、今後はグラス向けの開発も開始できます。
アプリを開発する

使用するフレームワーク（Android Jetpack XR、Unity、OpenXR、WebXR）を選択します。

使い慣れた Android 開発ツール、人気の高いゲームエンジン Unity、オープン スタンダードである OpenXR、幅広く利用可能な WebXR を使用して、次世代のコンピューティング向けのアプリを作成します。

Android Jetpack XR

ほとんどの Android アプリは、追加の開発作業なしで Android XR に対応しています。Android XR がデバイスの操作とライフサイクル管理を処理するため、デベロッパーはイノベーションに集中できます。
Jetpack XR SDK を使用して開発する

Unity

Unity のコンテンツ制作機能に完全にアクセスし、他のプラットフォームから Android XR にアプリを簡単に移行できます。パフォーマンス最適化ツール、大規模なアセットストア、強力なコミュニティを活用して、スムーズな開発を実現します。
Unity での開発

OpenXR

OpenXR の著作権フリーのオープン標準で開発を効率化します。共通の API セットを使用してどこでもビルドし、さまざまなデバイスで動作する XR アプリを開発できます。
OpenXR で開発する

WebXR

ウェブ技術を活用して、ブラウザ内で直接 XR エクスペリエンスを構築できます。WebXR を使用すると、デバイスとサポートされているウェブブラウザがあれば、誰でも拡張現実を利用できます。
WebXR で開発する

Android XR を使ってみる

Android XR 向けの設計

Android XR 設計ガイドラインには、すぐに使えるデザイン システム、スタイル、UI スケーリング、コンポーネント、空間パネル、3D モデル、環境が含まれています。

XR の原則に従う

現状から始め、ユーザーが快適に利用できるようにし、XR で特別な体験を探求し、アプリがアクセス可能であることを確認します。

XR の基礎を学ぶ

Android XR を使用すると、高品質の XR アプリを迅速に構築できます。既存のアプリのコンテンツ、レイアウト、UI 要素を空間化するには、ビルド済みコンポーネントを使用します。

自然なやり取りを使用する

Android XR は、ユーザーに柔軟性と制御を提供します。ユーザーは、手や目のトラッキング、音声コマンド、Bluetooth デバイスを使用してアプリを操作できます。

空間フレームワーク、無限のディスプレイ空間、マルチモーダル入力を活用して、創造性と動的機能を無限に広げ、ビジョンを実現しましょう。

作成方法を簡素化する

2D アプリを XR に簡単に移行したり、3D エクスペリエンスに全面的に取り組んだりできます。信頼できるツールと業界をリードする API を使用して、迅速な開発を促進します。

Android で将来を見据えたスケーリング

Android XR のデバイス メーカーによるオープンなエコシステムと Google Play ストアの幅広いユーザーベースを活用して、アプリのリーチを拡大しましょう。
