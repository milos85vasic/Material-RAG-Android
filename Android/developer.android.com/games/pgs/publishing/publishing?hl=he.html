סקירה כללית על Play Games Services Publishing API
קל לארגן דפים בעזרת אוספים אפשר לשמור ולסווג תוכן על סמך ההעדפות שלך.

ה-Publishing API מאפשר לכם לבצע אוטומציה של משימות תכופות שקשורות להפקת משחקים ולהפצתם. ממשק ה-API הזה מספק פונקציות דומות לאלה שזמינות דרך Google Play Console, כמו:

  • יצירה ועריכה של רשימות הישגים.
  • יצירה ועריכה של רשומות בטבלת המובילים.
  • העלאת תמונות של סמלים להישגים ולטבלאות הישגים.

הערה: אפשר להשתמש ב-Publishing API יחד עם Google Play Developer API.

אפשר לגשת ישירות לממשקי ה-API ל-REST באמצעות HTTP. מידע נוסף זמין במאמר הפניה ל-API של פרסום ובאפליקציה לדוגמה.

דוגמאות התוכן והקוד שבדף הזה כפופות לרישיונות המפורטים בקטע רישיון לתוכן.‏ Java ו-OpenJDK הם סימנים מסחריים או סימנים מסחריים רשומים של חברת Oracle ו/או של השותפים העצמאיים שלה.
