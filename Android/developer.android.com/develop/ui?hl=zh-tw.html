開發 Android 使用者介面

Jetpack Compose 是 Android 推薦的新型工具包，可用來以較少的程式碼、強大的工具和符合直覺的 Kotlin API 建構原生 UI。這個平台可簡化及加快 UI 開發作業，包括根據智慧型手機、折疊式裝置、平板電腦、電視和穿戴式裝置等任何板型規格進行調整。

Compose 提供預先建立的 UI 元件，可以搭配使用圖像、動畫和其他視覺元素。

針對以 View 為基礎的現有 UI，您可以選擇遷移至 Compose，或繼續使用以檢視區塊為基礎的工作流程。
