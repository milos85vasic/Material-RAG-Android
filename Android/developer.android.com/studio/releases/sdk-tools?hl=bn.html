SDK টুল রিলিজ নোট
সেভ করা পৃষ্ঠা গুছিয়ে রাখতে 'সংগ্রহ' ব্যবহার করুন আপনার পছন্দ অনুযায়ী কন্টেন্ট সেভ করুন ও সঠিক বিভাগে রাখুন।

দ্রষ্টব্য: এই SDK টুলস প্যাকেজটি বাতিল করা হয়েছে এবং আর আপডেট পাচ্ছে না। পরিবর্তে, নতুন Android SDK কমান্ড-লাইন সরঞ্জাম প্যাকেজ ব্যবহার করুন৷ নতুন কমান্ড-লাইন টুলস প্যাকেজের আপডেটের জন্য, Android SDK কমান্ড-লাইন টুলস রিলিজ নোট দেখুন।

উপরন্তু, আমরা SDK ম্যানেজারের জন্য আর্টিফ্যাক্ট আপডেট করার একটি পুরানো উপায় বন্ধ করে দিচ্ছি। এই পরিবর্তনটি শুধুমাত্র অ্যান্ড্রয়েড স্টুডিও 2.2 এবং তার নিচের সংস্করণগুলিকে প্রভাবিত করে৷ সামনের দিকে, আমরা আর এই পুরানো XML ফর্ম্যাটের সাথে আপডেট প্রকাশ করব না। অ্যান্ড্রয়েড স্টুডিওর পুরানো সংস্করণ, পুরানো কমান্ড-লাইন SDK ম্যানেজার বা পুরানো SDK ম্যানেজার UI এর ব্যবহারকারীরা SDK ম্যানেজারের মাধ্যমে SDK উপাদানগুলির আপডেট পাবেন না৷ যদিও আপনার বর্তমান বিল্ডগুলি কাজ চালিয়ে যাওয়া উচিত, আপনি যদি SDK উপাদানগুলি আপডেট করতে চান তবে নতুন কমান্ড-লাইন সরঞ্জামগুলি ব্যবহার করুন৷

আপনি যদি অ্যান্ড্রয়েড স্টুডিও 2.2 বা তার নিচের সংস্করণ ইনস্টল করেন, তাহলে আপনি উপাদানগুলি পুনরায় ডাউনলোড করতে পারবেন না। SDK ম্যানেজারের মাধ্যমে উপাদানগুলি ডাউনলোড করতে, Android Studio-এর একটি নতুন সংস্করণে আপগ্রেড করুন৷

Android SDK টুলস হল Android SDK-এর একটি উপাদান। এতে Android এর জন্য ডেভেলপমেন্ট এবং ডিবাগিং টুল রয়েছে।

  • এখন প্ল্যাটফর্ম এবং সরঞ্জামগুলির উপাদানগুলির মধ্যে এবং এসডিকে অ্যাড-অন এবং প্ল্যাটফর্মগুলির মধ্যে নির্ভরতা প্রয়োগ করে।
লেআউটপট, লেআউটগুলি অনুকূলকরণের জন্য একটি নতুন সরঞ্জাম:

এসডিকে সরঞ্জাম আর 3 প্যাকেজটিতে layoutopt অন্তর্ভুক্ত রয়েছে, একটি নতুন কমান্ড-লাইন সরঞ্জাম যা আপনাকে আপনার লেআউট শ্রেণিবদ্ধকরণগুলি অনুকূল করতে সহায়তা করে। যখন আপনার লেআউট ফাইলগুলির বিরুদ্ধে চালানো হয়, সরঞ্জামটি তাদের শ্রেণিবিন্যাস বিশ্লেষণ করে এবং আপনাকে অদক্ষতা এবং অন্যান্য সম্ভাব্য সমস্যা সম্পর্কে অবহিত করে। সরঞ্জামটি এটি পাওয়া সমস্যাগুলির জন্য সহজ সমাধানও সরবরাহ করে। ব্যবহারের জন্য, লেআউটপট দেখুন।
