Supportpersonalisierung
Mit Sammlungen den Überblick behalten Sie können Inhalte basierend auf Ihren Einstellungen speichern und kategorisieren.

Wenn Sie ein einfaches Zifferblatt erstellt haben, können Sie den Nutzern die Möglichkeit geben, sowohl das Aussehen des Zifferblatts als auch die darauf angezeigten Informationen anzupassen. So wird die Smartwatch persönlicher und nützlicher.

Verwenden Sie UserConfigurations für die Personalisierung, damit Nutzer das Erscheinungsbild anpassen können. Wenn Sie die angezeigten Informationen anpassen möchten, müssen Sie Zusatzfunktionen unterstützen.

Bei beiden Arten von Anpassungen verwendet der Nutzer den Zifferblatteditor, um Änderungen vorzunehmen. Wenn Sie den Editor aktivieren möchten, müssen Sie das Zifferblatt bearbeitbar festlegen. Weitere Informationen finden Sie in der Verpackungsanleitung.
