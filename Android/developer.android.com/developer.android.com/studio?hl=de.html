Android Studio

Die offizielle IDE für die Entwicklung von Android-Apps steigert jetzt Ihre Produktivität mit Gemini in Android Studio, Ihrem KI-gestützten Programmierassistenten.

Ihr aktuelles Gerät wird nicht unterstützt.

Gemini in Android Studio ausprobieren

Gemini in Android Studio ist ein KI‑Assistent, der Sie dabei unterstützt, Code zu generieren, Code zu korrigieren und Fragen zur Entwicklung von Android-Apps zu beantworten.

Gemini in Android Studio ausprobieren
Funktion

Überall mit Android Studio Cloud programmieren

Sie können Android Studio Cloud direkt in Ihrem Browser ausprobieren, indem Sie über Firebase Studio darauf zugreifen. Optimieren Sie Ihren Workflow und überspringen Sie lokale Installationen. Testen Sie die Vorabversion.

Android Studio Cloud ausprobieren

Hauptfunktionen

Design

Designtools zum Erstellen von Inhalten

Mit Jetpack Compose dynamische Layouts erstellen Anschließend können Sie sich eine Vorschau Ihrer Layouts auf Displays beliebiger Größe ansehen und Compose-Animationen mit den integrierten Tools prüfen.
Design mit Compose
Entwickeln

Intelligenter Code-Editor

Mit einem intelligenten Code-Editor, der Codevervollständigung für die Programmiersprachen Kotlin, Java und C/C++ bietet, können Sie besseren Code schreiben, schneller arbeiten und produktiver sein. Außerdem können Sie bei der Bearbeitung von Jetpack Compose mit Live Edit sofort sehen, wie sich Ihre Codeänderungen auswirken.
Weitere Informationen zum Editor
Build

Flexibles Build-System

Das Build-System von Android Studio basiert auf Gradle und ermöglicht es Ihnen, Ihren Build so anzupassen, dass aus einem einzigen Projekt mehrere Buildvarianten für verschiedene Android-Geräte generiert werden. Anschließend können Sie mit dem Build Analyzer die Leistung Ihrer Builds analysieren und herausfinden, wo es in Ihrem Projekt potenzielle Build-Probleme gibt.

Mehr über Android Build
Test

Geräte ganz einfach emulieren

Mit dem Android-Emulator können Sie Ihre App auf einer Vielzahl von Android-Geräten testen. Mit responsiven Layouts können Sie das volle Potenzial Ihrer Apps ausschöpfen. Sie lassen sich an Smartphones, Tablets, faltbare Geräte, Wear OS-, TV- und ChromeOS-Geräte anpassen.

Android-Emulator verwenden
Veröffentlichen

Android App Bundle

Finde Möglichkeiten, die Größe deiner Android-App vor der Veröffentlichung zu optimieren, indem du den Inhalt deiner APK-Datei oder deines Android App Bundles prüfe. Untersuchen Sie die Manifestdatei, die Ressourcen und die DEX-Dateien. Vergleiche zwei APKs oder Android App Bundles, um zu sehen, wie sich die Größe deiner App zwischen den App-Versionen verändert hat.
