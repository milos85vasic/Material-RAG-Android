Android Studio の Gemini の機能
コレクションでコンテンツを整理 必要に応じて、コンテンツの保存と分類を行います。

Gemini in Android Studio には、開発プロセスのあらゆるステップに対応する機能が用意されています。Gemini は急速に発展している分野であるため、Android Studio の安定版リリースノートとプレビュー版リリースノートで最新の更新情報もご確認ください。

カテゴリ 機能 説明 ドキュメント
チャット チャット Gemini を操作するための元のインターフェース。開発ワークフローのどの時点でも、Android 関連の質問を Gemini に尋ねることができます。 Gemini と話そう
ファイルを添付 クエリにファイルを添付すると、Gemini が質問のコンテキストをより的確に把握し、更新が必要なコードを特定できます。 ファイルを添付する
画像を添付する クエリに画像を添付すると、UI の生成、UI バグのデバッグ、アプリのアーキテクチャの文書化などに役立ちます。

無料枠のみ。		 画像を添付する
プロンプト ライブラリ（IDE 内） よく使用するプロンプトを保存して管理し、必要に応じてすばやく呼び出すことができます。 プロンプト ライブラリでプロンプトを保存して管理する
AGENTS.md ファイルのサポート コードベースの一部である 1 つ以上のマークダウン ファイルで、クエリに対する Gemini のレスポンスの優先順位を定義します。AGENTS.md ファイルで定義された手順は、IDE に依存しません。

プレビュー版。		 AGENTS.md ファイルを使用して Gemini をカスタマイズする
ルール クエリに対する Gemini の回答の優先度を定義します。優先するコーディング言語、スタイル、出力形式などを定義します。 ルールを使用してエクスペリエンスをカスタマイズする
プロンプト ギャラリー プロンプトのコレクションを参照して、Gemini をワークフローに組み込む方法のヒントを得ましょう。 プロンプト ギャラリー
ローカル LLM パソコンにローカルにインストールされている LLM を選択して、Android Studio の AI 機能を強化します。

プレビュー版。		 ローカル LLM を使用する
外部ツールとの統合 Android Studio を離れることなく、外部ツールに関連する情報を取得してアクションを実行します。@ と入力してツールの一覧を表示し、Gemini のプロンプトに含めるツールを選択します。たとえば、「@GoogleDocs 最近のドキュメントを一覧表示して」と入力して、最近の Google ドキュメントの一覧を表示するよう Gemini にリクエストできます。

エージェント モード エージェント モード エージェント モードは、Gemini とチャットするだけでは対応しきれなかった、複雑なマルチステージの開発タスクを処理するように設計されています。エージェントは複数のファイルにわたって変更を行い、バグを繰り返し修正できるため、開発プロセスを加速できます。 エージェント モード
API キーを追加する エージェント モードの場合: API キーを追加してコンテキスト ウィンドウを拡大し、さらに高品質のレスポンスを取得します。

無料枠のデフォルト モデルのユーザーにのみ適用されます。		 独自の Gemini API キーを追加する
MCP サーバーを追加する エージェント モードの場合: Model Context Protocol（MCP）を使用して、外部ツールとやり取りし、知識と機能を拡張します。 MCP サーバーを追加する
コード コード補完 Gemini は、Android Studio で AI によるコードの予測入力機能を提供します。入力時に、グレーの斜体テキストとして表示されます。コード補完では、完全な関数が提案されるため、コーディング プロジェクトをより迅速に完了できます。 AI コード補完でコーディングを加速する
コード変換 コードエディタから Gemini にプロンプトを送信して、プロジェクトのコードの変更、最適化、追加を行うためのコード提案を取得します。 変換コード
変数名を再検討する Gemini から、ファイル内の 1 つまたはすべての変数の名前を変更する方法についての提案を受け取ります。 変数名を再検討する
単体テスト シナリオを生成する Gemini は、テストするコードのコンテキストを使用してテスト シナリオを提案できます。単体テスト シナリオを生成する際、Gemini はテストの詳細な名前と説明を含めるため、提案された各テストの意図をより深く理解できます。 単体テスト シナリオを生成する
Gemini のジャーニー 自然言語を使用してステップとアサーションを記述することで、ジャーニーと呼ばれるエンドツーエンドの機能テストを作成します。Gemini は、手順を Gemini がアプリで実行するアクションに変換します。

プレビュー版。		 Android Studio のジャーニー
コードの文書化 コードの明確かつ簡潔なドキュメントを生成し、下書きをすぐに作成して、それを洗練して完成させることができます。 コードのドキュメントを生成する
commit メッセージを生成する Gemini は、コード変更と最近の過去のコミットのコンテキストを使用して、コミット メッセージを生成できます。 コミット メッセージを生成する
コードのカスタマイズ コードのカスタマイズを使用すると、組織のプライベート リポジトリに基づいて、組織のコーディング スタイルに準拠したコード候補を取得できます。

Business ティアのサブスクライバーのみ。		 コードのカスタマイズの概要
作成 Compose プレビューの生成 Gemini は、特定のコンポーザブルまたはファイル内のすべてのコンポーザブルについて、プレビュー パラメータのモックデータを含む Compose プレビューを自動的に生成できます。

Studio Labs で有効にします。		 Compose プレビューを生成する
変換 UI 自然言語を使用して、Compose プレビュー パネルからアプリの UI を直接更新します。

Studio Labs で有効にします。		 変換 UI
Android を搭載した統合 App Quality Insights でクラッシュを分析する Gemini を使用して、アプリ品質分析情報のクラッシュ レポートを分析し、分析情報を生成して、クラッシュの概要を提供します。また、可能な場合は、サンプルコードや関連ドキュメントへのリンクなど、次のステップを推奨します。 App Quality Insights と Gemini でクラッシュを分析する
Logcat でランタイム エラーを分析する Android Studio の Gemini を使用すると、[Logcat] ウィンドウのエラーを理解して解決できるため、デバッグ プロセスを効率化できます。アプリでエラーや例外が発生した場合は、[Gemini に質問] をクリックすると、IDE を離れることなく、すぐに説明と実行可能な提案を取得できます。 Logcat と Gemini を使用してランタイム エラーを分析する
ビルドエラーと同期エラーに関するヘルプ Gemini は Gradle のビルドエラーと同期エラーを理解します。エラーが発生した場合は、ビルド出力の [Gemini に質問] リンクをクリックして、問題の診断と修正に役立ててください。
プライバシーとセキュリティ コンテキストの共有を構成する .aiexclude ファイルを使用して、Gemini と共有するファイルを具体的に構成します。 コンテキスト共有を構成する
ロギング プロンプトとレスポンス、ユーザーが承認したコード行などのメタデータを含む Gemini アクティビティを Cloud Logging で収集します。
