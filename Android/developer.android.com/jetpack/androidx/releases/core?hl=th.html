เวอร์ชัน 1.0.2

7 พฤษภาคม 2019

androidx.core:core:1.0.2 และ androidx.core:core-ktx:1.0.2 เปิดตัวแล้ว รุ่นนี้มีการคอมมิต 2 รายการ

การเปิดตัว AndroidX ครั้งแรกทำให้การรองรับการทำงานย้อนหลังสำหรับ Support Library compat API ที่ใช้สำหรับการสื่อสารระหว่างกระบวนการและการส่งข้อมูลที่ใช้คีย์ ใน Bundle หยุดทำงาน การเปิดตัวนี้แก้ไขความเข้ากันได้ดังกล่าว ซึ่งจะช่วยให้การสื่อสาร ระหว่างโค้ดที่สร้างด้วย Support Library 28.0 (หรือก่อนหน้า) กับ AndroidX 1.1 (หรือใหม่กว่า) กลับมาเป็นปกติ

แก้ไขข้อบกพร่อง

  • ใช้ค่าสตริงก่อนเวอร์ชัน 1.0 สำหรับ InputConnectionCompat (aosp/948725)
  • ใช้สตริงก่อน AndroidX เป็นการทำงานร่วมกันสำรองสำหรับคีย์ของ Bundle (aosp/938269)
